Jak połączyć css z kodem html

CSS (ang. Cascading Style Sheets) to język służący do opisu prezentacji dokumentów HTML i XML. Plik z CSS zapisujemy z rozszerzeniem .css i podłączamy go do głównego pliku html za pomocą znacznika "link rel="stylesheet" type="text/css" href="nazwa_pliku.css"".

Selektory

Selektory to elementy pozwalające na wybór elementów do zmiany ich stylów. Przykłady selektorów to:

  • id - pozwala na wybór elementu po jego id
  • klasa - pozwala na wybór elementów po ich klasie
  • element - pozwala na wybór wszystkich elementów danego typu

Właściwości

Właściwości to cechy elementów, które możemy zmieniać za pomocą CSS. Przykłady właściwości to:

  • color - kolor tekstu
  • font-size - rozmiar czcionki
  • background-color - kolor tła
