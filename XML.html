Kurs XML

Podstawy

XML (ang. Extensible Markup Language) to język znaczników służący do tworzenia i przesyłania danych. Jest on rozszerzeniem języka HTML i pozwala na tworzenie własnych znaczników.

Tworzenie dokumentu

Aby utworzyć dokument XML, należy użyć nagłówka oraz znacznika . Poniżej znajduje się przykład prostego dokumentu XML:

<?xml version="1.0"?>
<note>
  <to>Tove</to>
  <from>Jani</from>
  <heading>Reminder</heading>
  <body>Don't forget me this weekend!</body>
</note>

Znaczniki

Znaczniki XML składają się z nazwy znacznika oraz jego zawartości. Nazwa znacznika jest unikalna dla danego dokumentu. Poniżej znajduje się przykład znacznika:

content

Atrybuty

Atrybuty to dodatkowe informacje przypisane do znaczników. Są one umieszczane w nawiasach po nazwie znacznika. Poniżej znajduje się przykład znacznika z atrybutem:

content

Dokumenty z wyłącznie elementami

XML pozwala na tworzenie dokumentów złożonych tylko z elementów. W tym przypadku nie ma potrzeby używania atrybutów, ponieważ informacje są przekazywane przez elementy.

Narzędzia

Aby korzystać z języka XML potrzebujemy narzędzi do jego edycji, np. Notatnika++, Sublime Text. Do tworzenia i walidacji dokumentów XML możemy użyć narzędzia jak np. XML Copy Editor, jEdit, XML Notepad.
